# 🚀 LifeBalance App

A full-stack multi-feature web application.

## 🔥 Features
- Authentication (JWT)
- Expense Tracker (DB connected)
- Calculator
- Percentage Tool
- Pacman Game

## 🛠 Tech Stack
- Node.js, Express
- MongoDB Atlas
- HTML, CSS, JS

## 📦 Run Locally
cd server
npm install
node server.js

Open client/index.html
